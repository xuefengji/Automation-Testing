### web基础

##### 网络标准体系架构

+ B/S：browser/server
+ C/S：client/server
+ P2P：point to point

B/S   vs C/B ：

+ 标准和规范
  + B/S是标准规范的，C/S是不标准的
+ 运算能力：
  + B/S：核心运算都在服务端
  + C/S：客户端和服务端都可以承担运算
+ 部署：
  + B/S：部署方便，只需部署服务端
  + C/S：部署很麻烦，需要同时升级客户端和服务端
